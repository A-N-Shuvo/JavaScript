# JavaScript Step-by-Step Guideline (Action Plan)

## Step 1: Setup (Day 1)
- Install Node.js LTS
- Install VS Code + extensions (ESLint, Prettier)
- Create folder: `js-learning-lab`
- Create first file: `index.js`
- Run with: `node index.js`

## Step 2: First 7 Days (Syntax Confidence)
- Day 1: variables, primitive types, `typeof`
- Day 2: operators and type coercion
- Day 3: conditionals and loops
- Day 4: functions and return values
- Day 5: arrays and array loops
- Day 6: objects and nested objects
- Day 7: mini revision + build simple quiz app

## Step 3: Week 2-3 (Logic + Reusability)
- Practice array methods daily
- Solve 3 algorithmic tasks/day (easy)
- Learn object destructuring and spread
- Build utilities: capitalize, validator, formatter

## Step 4: Week 4-5 (DOM + Events)
- Select and update elements
- Handle events and forms
- Event delegation for dynamic lists
- Build Todo app with edit/delete/filter

## Step 5: Week 6-7 (Async + APIs)
- Understand callback -> promise -> async/await progression
- Consume public APIs (`fetch`)
- Add loading and error states
- Build weather or movie search app

## Step 6: Week 8-9 (Deep JavaScript)
- Closures with practical counters
- Scope and hoisting exercises
- `this` in regular vs arrow functions
- Prototypes and class-based models

## Step 7: Week 10-11 (Node + Backend)
- Setup Express server
- Build CRUD endpoints
- Add middleware and validation
- Connect frontend to backend

## Step 8: Week 12+ (Project + Polish)
- Plan features and architecture
- Implement modules one by one
- Add tests and documentation
- Deploy and share

## Daily Checklist
- Write code without copy-paste for 30+ minutes
- Explain one concept in your own words
- Commit code with clear messages
- Track one improvement for next day

## Weekly Checklist
- Build one mini-project
- Attempt 50 MCQs from `MCQ-Bank`
- Refactor one old project file
- Record what you learned and what blocked you
