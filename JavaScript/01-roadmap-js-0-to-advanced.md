# JavaScript Roadmap (Beginner to Expert)

## Phase 1: Beginner Foundation (Weeks 1-3)
- JavaScript basics: variables, data types, operators
- Control flow: if/else, switch, loops
- Functions: declaration, expression, arrow functions
- Arrays and objects
- Scope basics (`var`, `let`, `const`)
- Basic DOM selection and events

### Deliverables
- Build: Calculator, Todo (local only), Counter app

## Phase 2: Core Skill Building (Weeks 4-6)
- Array methods: `map`, `filter`, `reduce`, `find`, `some`, `every`
- String and object utilities
- Error handling: `try/catch/finally`
- ES6+ essentials: destructuring, spread/rest, template literals
- Modules: import/export
- JSON and browser storage (`localStorage`, `sessionStorage`)

### Deliverables
- Build: Expense tracker with localStorage

## Phase 3: Async + API + Browser (Weeks 7-9)
- Event loop, call stack, task queue, microtask queue
- Promises and `async/await`
- `fetch` API and HTTP methods
- Form validation and error-first UX
- Debounce and throttle
- Browser concepts: event bubbling, capturing, delegation

### Deliverables
- Build: Weather app, GitHub profile finder, Notes app with API

## Phase 4: Advanced JavaScript (Weeks 10-12)
- Closures, lexical scope, hoisting deep-dive
- `this`, `bind`, `call`, `apply`
- Prototype chain and inheritance
- Classes and OOP in JavaScript
- Functional concepts and immutability
- Memory leaks and performance basics

### Deliverables
- Build: Reusable UI components library (vanilla JS)

## Phase 5: Node.js + Tooling + Testing (Weeks 13-15)
- Node runtime, modules, npm scripts
- Express basics and REST API
- Middleware and centralized error handling
- Environment variables and config
- Testing basics: unit/integration concepts
- Linting and formatting (`eslint`, `prettier`)

### Deliverables
- Build: Task API + frontend integration

## Phase 6: Expert Path (Weeks 16+)
- Architecture and folder design patterns
- Performance profiling and optimization
- Security basics (XSS, CSRF, sanitization)
- CI/CD basics and deployment
- Documentation and maintainability practices

### Deliverables
- Build: Full capstone project from scratch

## Capstone Project Suggestions
- Full-stack task manager
- Job board with auth and admin panel
- E-commerce mini-store
- Learning management portal

## What to Learn Daily
- 40% concept reading
- 50% coding practice
- 10% revision and notes
